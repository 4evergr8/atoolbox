## 📌 项目简介 / About

PicOrigin 是一款基于 Flutter 构建的跨平台图片工具，专注于**图片出处查找与反向搜图（Reverse Image Search）**。

PicOrigin is a cross-platform Flutter application focused on **image source finding and reverse image search**.

用户可以快速通过多种搜索引擎查找图片来源、相似图片或原始出处。

Users can quickly find image sources, similar images, and original references using multiple reverse image search engines.

## 🔍 支持的反向搜图引擎 / Supported Reverse Image Search Engines

PicOrigin 聚合多个主流“以图搜图 / 找出处”引擎：

PicOrigin integrates multiple reverse image search engines:

- Google Images（综合搜图 / general search）
- Google Lens（图像识别 / visual search）
- Yandex Images（动漫/人物识别强 / strong for anime & people）
- Bing Visual Search（微软图像搜索）
- TinEye（经典反向搜图 / classic reverse search）
- SauceNAO（动漫 / 插画 / 漫画 / anime & illustration）
- IQDB（动漫/插画作品来源 / anime artwork source finder）
- 3DIQDB（Cosplay / 3D相关）
- ascii2d（日本二次元作品来源）
- trace.moe（动漫截图识别 / anime scene finder）
- Lenso.ai（AI图像识别 / AI-powered reverse image search）


👉 一键打开多个引擎，提高“找出处”成功率  
👉 One-click multi-engine search to maximize source detection accuracy

## ✨ 核心功能 / Features

### 🔎 1. 以图搜图 / Reverse Image Search
支持本地图片或图片URL输入，快速跳转多个搜索引擎查找来源。

Supports local images or image URLs and redirects to multiple reverse search engines.

---

### 📷 2. 图片来源查找 / Image Source Finder
用于查找：

- 图片原始出处
- 作者或画师信息
- 相似图片
- 网络传播源头

Find:

- Original image source
- Artist or creator
- Similar images
- Web origin tracking

---

### 📱 3. 分享即搜 / Share-to-Search
从系统分享图片直接进入应用，一键进行反向搜图或识别。

Share images from other apps directly to PicOrigin for instant search.

---

### 📊 4. 多引擎聚合 / Multi-Engine Aggregation
一次搜索同时覆盖多个搜索平台，提高查找成功率。

Search across multiple engines simultaneously to improve accuracy.

---

### 🌐 5. 跨平台支持 / Cross Platform
基于 Flutter 构建，可运行于：

- Android
- iOS（需自行构建）
- Web（可选）

Built with Flutter, supporting Android, iOS, and Web.

## 🚀 使用方法 / How to Use

### 📌 方法一 / Method 1
1. 打开应用  
   Open the app  
2. 选择图片  
   Select an image  
3. 点击“以图搜图”  
   Tap “Reverse Image Search”  
4. 自动打开多个搜索引擎结果  
   Multiple search engines will be opened

---

### 📌 方法二 / Method 2
1. 在任意应用中分享图片  
   Share an image from any app  
2. 选择 PicOrigin  
   Select PicOrigin  
3. 自动进入分析页面  
   Enter analysis screen  
4. 一键搜索出处  
   Search image source instantly
