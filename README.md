    图片搜索，离线翻译，文字识别，网络测速，DNS查询，扫码识别，内网扫描，收藏备份，视频下载

## 主要功能
### 在线功能
* 搜索本地图片和网络图片
* 搜索哔哩哔哩视频封面
* 备份哔哩哔哩收藏夹
* 批量下载哔哩哔哩视频
* 下载大文件测试网速
* 加密DNS查询
### 离线功能
* URL解码
* Base64编码解码
* AVBV互转
* 离线扫码
* 扫描局域网设备
* 扫描设备端口
* 图片文字提取（OCR）
* 离线翻译（支持中、英、日、韩、俄互译，需要下载模型）
