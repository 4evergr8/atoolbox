## 主要功能
### 分享功能

* 搜索接收到的图片
* 对收到的图片进行OCR
* 识别图片中的二维码
* 分享视频备份
* 搜索分享视频的封面

### 在线功能

* 搜索本地图片和网络图片
* 搜索哔哩哔哩视频封面
* 备份哔哩哔哩视频,封面,简介
* 下载大文件测试网速
* 加密DNS查询测试
* IP反查域名

### 离线功能

* URL解码编码
* Base64编码解码
* 兽音译者解码
* 离线扫码
* 穷举乱码恢复(demo)
* AV号BV号转换
* 多语言文本识别(OCR)

## 编译

* 本项目采用GithubAction进行编译
* Fork或Import本项目,在Action选项卡内运行Build and Release Flutter APK  
* 等待成功运行完成后在Release里寻找最新测试版并下载对应架构的APK
